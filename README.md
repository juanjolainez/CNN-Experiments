# CNN-Experiments

CNN experiment to solve the Street View House Number (SVHN) problem.

The code is some modification of the code available in the course 

Deep Learning Convolutional Neural Networks with Theano and Tensorflow
https://udemy.com/deep-learning-convolutional-neural-networks-theano-tensorflow

The dataset is the following: 

http://ufldl.stanford.edu/housenumbers/

Download the 32x32 datasets and store it in the a separate folder called 
large_files. 
From the file the route will be `../large_files/train_32x32.mat`

This is just a way to keep track of some changes done to the original document.
